HTML、CSS、Javascript、Citescape.jsを利用してネットワーク図を作成するツールを作成してください。

# 画面構成
- 画面上部にメニューバーを設定
　メニューには「File」、「Style」、「Filter」、「Layout」を表示させる（メニューは今後、増やす可能性あり）
　※今回は主に「File」メニューのみを設定、他のメニューの機能はあとで追加予定
- 画面全体をネットワーク表示領域にする
- シンプルだが見やすくデザイン性を持たせる

# 「File」メニュー
- 「File」をクリックすると「Import」→「Network File」と「Table File」メニューを表示させる
- 「Network File」と「Table File」をクリックするとファイルダイアログを開きファイルを選択できるようにする

# 取込むファイル
- ネットワークファイル(エッジファイル)　ファイル種別：CSV、Excel
- テーブルファイル(ノードファイル)　ファイル種別：CSV、Excel

# 「Network File」メニュー
- 選択したファイルからカラムを読込み、どのカラムをSource、Target、attributeにするか設定できるようにする
　(デフォルトでは1カラム目をSource、2カラム目をTarget、他をattributeに設定)
- attributeに設定するときDataTypeを設定できるようにする
　(文字列、数値、浮動小数点、Y/N、およびそれぞれの型の配列、配列時の区切り文字)※デフォルトは文字列
attribute
# 「Table File」メニュー
- 選択したファイルからカラムを読込み、どのカラムをNode、attributeにするか設定できるようにする
　(デフォルトでは1カラム目をNode、他をattributeに設定)
- attributeに設定するときDataTypeを設定できるようにする
　(文字列、数値、浮動小数点、Y/N、およびそれぞれの型の配列、配列時の区切り文字)※デフォルトは文字列

# レイアウト
- レイアウトのデフォルトは「Hierarchical」

不明点があれば質問してください
　
==========================================================================
・Network File SettingとNode File Settingのウインドウでは「Data Preview」は不要です。
・「Column Settings」では読み込んだカラムが縦表示ですが横表示にして各設定を縦に並べるようにしてください。
・読込んだデータでカラムの値が全て整数値の場合は、自動でintegerに設定するようにしてください
・「Import」ボタンを押下したときにファイルが大きいと読み込みに時間が掛かる場合がありますが、何のアクションもないため
処理されているか不安になるため「Import」ボタンを押下したときにボタンが凹むようにし処理中などのメッセージを出力するようにしてください。
==========================================================================
・Network File SettingとNode File SettingのColumn Settingsでは表形式にしてください(省スペースで表示)
・Column SettingsのSource、TargetにはDATA TYPEとDELIMITERは不要です。
不明点があれば質問してください
==========================================================================
「Layout」メニューから「Layout Tools」メニューを作成してください
「Layout Tools」メニューを選択すると「Layout Tools」ウインドウが開くようにしてください
「Layout Tools」の機能に以下を作成してください
・Scale機能
　ネットワーク図を選択した項目によってスライドバーをスライドさせることによって引き延ばしたり縮めたりする機能
　項目「Width」、「Height」、「Selected Only」のチェックボックスを作成し
　その下に1/8,1/4,1/2,1,2,4,8のメモリの付いたスライドバーを作成
・Rotate機能
　-180,-90,0,90,180のメモリの付いたスライドバーを作成
　スライドバースライドさせることによってネットワーク図を回転させる
==========================================================================
「Layout Tools」ウインドウにて「Scale」を設定したあとに「Rotate」を設定しようとすると「Scale」で設定したものがデフォルトに戻ってしまします
「Rotate」を設定したあとに「Scale」を設定するときも同様に「Rotate」で設定したものがデフォルトに戻ってしまします。
「Scale」を設定を維持しつつ「Rotate」を設定できるようにしてください
また「Rotate」を設定を維持しつつ「Scale」を設定できるようにしてください
